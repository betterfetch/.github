# Installation
To install betterfetch, run this command

```bash
curl -fsSL https://betterfetch.github.io/scripts/rs/install.sh | bash
```

# Run without installing
>[!WARNING]
>This might be slower then installing and then running it

```bash
curl -sSL https://betterfetch.github.io/scripts/rs/launch | bash
```

**See more at <https://betterfetch.github.io>**
