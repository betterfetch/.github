# To install the Rust version
1. clone the repo
  ```bash
  git clone https://github.com/betterfetch/betterfetch.git
  ```
2. install using cargo
   ```bash
  cd betterfetch
  cargo install --path .
  ```
3. Add to path  
  ```bash
  # Add this to your ~/.zshrc or ~/.bashrc
  export PATH="$HOME/.cargo/bin:$PATH"
  ```
